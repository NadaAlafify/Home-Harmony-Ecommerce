# Home Harmony - Home Accessories E-commerce 

##  Description
**Home Harmony** is a fully E-commerce frontend layout dedicated to home accessories and kitchen tools. This project showcases a clean, user-friendly interface with structured product listings and interactive elements.

##  Live Demo
Check out the live project here: (https://nadaalafify.github.io/Home-Harmony-Ecommerce/) 


##  Tech Stack
- **HTML5:** Semantic structuring for an organized web store layout.
- **CSS3:** Custom styles, modern UI component designs and layout alignment.
- **JavaScript (Vanilla):** Implemented for dynamic interactive features.

##  Key Features
- **Product Catalog:** Well-designed grid displaying products with images, prices, details, and action buttons.
- **Interactive Top Bar:** Clean navigation panel featuring a shopping cart counter `Cart (0)` and authorization triggers (`Login` & `Register`).
- **Search Functionality:** An integrated product search bar with custom filters.
- **Pure Code Foundation:** Written 100% manually from scratch to master DOM manipulation, logical styling, and pure frontend flow without external heavy frameworks.
